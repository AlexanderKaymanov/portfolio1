Про меня

Меня зовут Александр

Здравствуйте! Меня зовут Александр Кайманов. Образование высшее, техническое, инженер-электромеханик. Я начинающий фронтенд-разработчик. Прошел курсы «ВЕБ-разработчик 2021 - с нуля до результата», «JavaScript - полный курс с нуля до результата» и «Посадка верстки и создание тем на CMS WordPress» на платформе beONmax.com. Изучал верстку сайтов на обучающей платформе Яндекс.Практикум. В процессе обучения освоил HTML (HTML5), CSS (CSS3), SASS, SCSS, JavaScript, Webpack, Gulp, методологию БЭМ Nested, а также работу с репозиториями в Git. Не останавливаюсь на достигнутом, постоянно развиваюсь.

Владею навыками адаптивной, кросс-браузерной вёрстки по макетам в .psd и .figma. Добавляю страницам сайта интерактивность и анимацию. В рамках обучения выполнил несколько самостоятельных проектов, которые можно посмотреть в моём аккаунте на GitHub или на сайте в портфолио.

В данный момент нахожусь в поиске заказов, по интересной для меня специальности, на фрилансе.

Верстка сайтов
Адаптивная, кросс-браузерная вёрстка по макетам в .psd и .figma.
Разработка приложений
Создание на JS и JQuery интерактивных страниц и не сложных приложений по ТЗ заказчика.
Посадка верстки и создание темы на CMS WordPress
Выполняется, как один из этапов верстки сайта, по согласованию с заказчиком. Шаблоны тем WP не использую, только оригинальная верстка по макету.
